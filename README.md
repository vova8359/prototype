# Прототип OLX Бота

Прототип проекту для роботи з OLX через Telegram бота.

## Структура проекту

- `бот/` - Основний Telegram бот
- `авторизація/` - Модулі авторизації в OLX
- `База даних/` - Скрипти для роботи з базою даних
- `Парс/` - Модулі парсингу даних
- `Повідомлення/` - Модулі для роботи з повідомленнями
- `Проксі/` - Менеджер проксі
- `Оновлення токенів/` - Система оновлення токенів
- `mini_app/` - Міні-додаток для Telegram
- `сайт/` - Веб-інтерфейс
- `тест/` - Тестові скрипти

## Встановлення

1. Встановіть залежності:
```bash
pip install -r інше/requirements.txt
```

2. Налаштуйте `.env` файл з необхідними токенами

3. Запустіть бота:
```bash
python бот/main.py
```

## Примітки

- Проект знаходиться в стадії розробки
- Деякі функції можуть бути нестабільними
